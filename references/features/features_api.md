# Features API

This document outlines the features available in the Ascort Bali application, categorized by user role.

## Homepage (Public)

- **View Creator Listings:** Browse a grid of creator profiles.
- **Filter Creators:** Filter creators by various criteria such as nationality, orientation, age, height, and bust size.
- **Pagination:** Navigate through multiple pages of creator listings.
- **View Ad Placements:** See advertisements in designated slots (`home-1`, `home-2`, `home-3`, and bottom).
- **View Service Details:** Click on a creator's card to view their detailed service offerings.

## User

- **User Registration:** Create a new user account.
- **User Login:** Authenticate and access the user portal.
- **User Logout:** Terminate the current session.
- **Profile Management:** Update personal information.
- **View Order History:** See a list of past and current orders.
- **Manage Favorites:** Add or remove creators from a personal favorites list.
- **Book a Service:** Initiate an order for a creator's service.

## Creator

- **Creator Registration:** Create a new creator account.
- **Creator Login:** Authenticate and access the creator portal.
- **Creator Logout:** Terminate the current session.
- **Profile Management:** Manage public profile information, including bio, languages, and location.
- **Service Management:** Create, update, and manage service offerings, including descriptions, pricing, and images.
- **Image Management:** Upload and manage a gallery of up to 20 images.
- **View Orders:** See a list of orders for their services.

## Admin

- **Admin Login:** Authenticate and access the admin portal.
- **Admin Logout:** Terminate the current session.
- **Ad Management:** Manage homepage ad images, links, and text.
- **User and Creator Oversight:** View counts and manage user and creator accounts.
- **Analytics:** View analytics data related to user activity and platform usage.
